Webhook Setup Instructions
1. Environment Variables

Set these environment variables in your deployment platform:

TELEGRAM_BOT_TOKEN=your_bot_token_here
SESSION_SECRET=your_session_secret_here

3. Set Webhook Command

Run this command to set the webhook:

curl -F "url={{webhook_url}}" https://api.telegram.org/bot{{bot_token}}/setWebhook
Pro Tip: After setting the webhook, test your bot by sending /start command in your Telegram bot chat.
